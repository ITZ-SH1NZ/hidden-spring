# HIDDEN SPRING
***An Atmospheric Easter Adventure***

---

## OVERVIEW

Hidden Spring is a whimsical yet structured Easter journey. With brutalist styling fused with interactive modern web mechanics, this project pushes the boundaries of atmospheric exploration in the browser. Players navigate through several intense levels, unlocking lore, dodging hazards, and ultimately uncovering the Golden Egg.

## ARCHITECTURE

Built to be fast, responsive, and deeply atmospheric, leveraging a robust modern stack to deliver seamless audio-visual experiences.

*   **Framework** Next.js App Router
*   **Styling** Tailwind CSS
*   **Icons** Phosphor Icons & Lucide React
*   **Motion** Framer Motion

---

## CORE MECHANICS

### Immersive Navigation
Smooth scrolling, custom cursors, and spatial audio visualizers create a tense, engaging web experience.

### Level Progression
*   **Level 1 | The Descent:** A test of timing and precision.
*   **Level 2 | Stealth:** Radar-based tactical navigation.
*   **Level 3 | The Run:** Fast-paced twitch mechanics.
*   **Level 4 | The Forge:** Puzzle-solving elements.
*   **Level 5 | The Boss:** A climactic battle for the Golden Egg.

---

## LOCAL DEVELOPMENT

To experience the journey locally, initialize the development environment.

**1. Install Dependencies**
```bash
npm install
```

**2. Start the Server**
```bash
npm run dev
```

**3. Enter the Spring**
Navigate to `http://localhost:3000` in your preferred web browser.
